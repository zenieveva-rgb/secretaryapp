Key things to remember

!

signup.html needs script.js

Add <script type="module" src="script.js"></script> before </body> in signup.html or the form will do nothing.

!

Firebase rules must allow writes

Default rules block all writes. Set ".write": true in Firebase Console → Realtime Database → Rules while testing.

i

approved must be boolean true

The login check uses user.approved. If the field is missing, is the string "true", or is false, login will be denied.

i

Camera requires HTTPS

The browser blocks camera access on plain http://. Use localhost locally, or deploy to Firebase Hosting for HTTPS.

QR Code Format

Student QR codes must follow this exact format. The app splits by | to extract each field.

Required format
LRN|StudentName|Grade
Example
123456789012|Juan Dela Cruz|Grade 7
Field 1LRN (12-digit learner reference number)
Field 2Full name of the student
Field 3Grade level (e.g. Grade 7, Grade 10)
